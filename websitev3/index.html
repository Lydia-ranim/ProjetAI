Comment utiliser LYHLYH

1️⃣
Définir les points : Cliquez sur la carte ou tapez le nom d'une station d'Alger (ex: Tafourah).
2️⃣
Choisir l'IA : A* est le plus équilibré. Dijkstra garantit le chemin absolu. Bi-Dir est le plus rapide en exécution.
3️⃣
Profils : Ajustez vos priorités dans les paramètres (Temps vs Argent vs CO₂).
4️⃣
Résultats : Analysez les coûts, l'empreinte carbone et suivez la ligne animée sur la carte.
